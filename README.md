# React Component Library Demo

This project demonstrates how to use the components from:

@lauraadan/react-component-library

## Installation

Clone the repository:

git clone https://github.com/lauraadan/react-component-library-demo.git

Install dependencies:

npm install

Run the development server:

npm run dev

The app will start at:

http://localhost:5173

## Components included

The demo showcases the following components:

- Button
- Alert
- Card
- Accordion
